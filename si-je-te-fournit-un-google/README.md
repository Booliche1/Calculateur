# Dofus Spell Tools

Site statique personnel pour suivre les relances de sorts Dofus et calculer les dégâts.

## Pages

- `index.html` : suivi des relances
- `calculateur.html` : calculateur de dégâts

## Lancer en local

```bash
npm run dev
```

Puis ouvrir `http://127.0.0.1:8765`.

## Déployer sur Vercel

1. Créer un dépôt GitHub.
2. Pousser ce dossier dans le dépôt.
3. Dans Vercel, créer un nouveau projet et importer le dépôt GitHub.
4. Laisser les réglages par défaut : aucun framework, aucune commande de build, dossier de sortie vide.

Les fichiers HTML, CSS et JS sont servis directement.

## Données

Les données de sorts proviennent de DofusDB. Utilisation prévue pour un projet personnel non commercial.
